## Quick start

```bash
git clone https://github.com/AbdullahBakir97/Py-Tetris-Game.git
cd Py-Tetris-Game
python -m venv .venv
source .venv/bin/activate    # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
